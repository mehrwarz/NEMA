# Next.js Project Guidelines

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Commands
- **Dev:** `npm run dev`
- **Build:** `npm run build`
- **Lint:** `npm run lint`

## Conventions
- Use functional TypeScript components with explicit types.
- Keep server and client components properly separated (`'use client'`).
